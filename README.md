# Chess Teacher

This project implements a simple chess AI capable of making chess moves at various skill levels and providing natural language explanations for its decision making.

### How to run

**Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

**Getting a move prediction**

First, run test_model.py
   ```bash
   python test_model.py
   ```

The program will prompt you for a FEN chessboard state.

Here's the FEN for the starting chess position:
   ```bash
   rnbqkbnr/pppppppp/8/8/8/8/PPPPPPPP/RNBQKBNR w KQkq - 0 1
   ```

You can create your own chess board position by using Lichess.org's board editor:
https://lichess.org/editor
